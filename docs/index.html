MFDB Classifier Training

This site provides instructions to train custom QIIME2 classifiers using the MFDB dataset (OTU/feature table + metadata on Zenodo), by filtering to a subset (e.g., body site/region) and re-training a classifier.

  • MFDB database: https://mfdb.irm.uzh.ch/#/
  • Zenodo record: 10.5281/zenodo.18494561
  • Prediction workflow repo (optional): Predict-mock-samples-forensic-BFID
